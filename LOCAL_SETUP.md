# Локальное тестирование приложения

## Требования
- Node.js (версия 14 или выше)
- PostgreSQL база данных

## Установка зависимостей

```bash
npm install
```

## Настройка базы данных

1. Создайте файл `.env` в корне проекта:
```
DATABASE_URL=postgresql://username:password@localhost:5432/dbudget
SESSION_SECRET=your-secret-key-here
PORT=3000
```

2. Создайте базу данных PostgreSQL:
```sql
CREATE DATABASE dbudget;
```

Приложение автоматически создаст необходимые таблицы при первом запуске.

## Запуск приложения

```bash
npm start
```

Приложение будет доступно по адресу: http://localhost:3000

## Функционал

### Для администратора:
- Просмотр всех объектов ремонта
- Добавление/редактирование/удаление объектов
- Управление пользователями (создание/удаление)
- Назначение объектов клиентам

### Для клиента:
- Просмотр только назначенных объектов
- Просмотр бюджета и расходов
- Без возможности редактирования
